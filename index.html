Game Board Sandbox 🎲

Import PNG or JPEG images, drop them onto the board, drag them around like pieces, and save the whole scene into a shareable session URL.

1. Import game pieces

Each imported image becomes a draggable board piece automatically.

2. Session URL

Tip: this stores the session directly in the URL hash. Great for lightweight sessions, but very image-heavy boards can make the URL too large for some browsers.

3. Board tools

Drag pieces to move them. Click a piece to select it. Use Delete or Backspace to remove the selected piece.
Board surface: 2200 × 1400
Accepted files: PNG, JPEG
Session mode: URL hash
